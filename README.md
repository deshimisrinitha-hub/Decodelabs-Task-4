# DecodeLabs Task 4: IoT Automation Logic ⚙️

## Project Overview
This project demonstrates a basic IoT automation system using software simulation. It collects simulated sensor data and performs automatic actions based on predefined conditions.

The system monitors temperature values and triggers alerts or actions when the temperature crosses the set limit.

## Objective
- Understand IoT automation concepts
- Simulate sensor data
- Apply conditional logic
- Generate alerts automatically
- Demonstrate automated control actions

## Features
- Temperature sensor simulation
- Real-time data monitoring
- Automatic alert generation
- Automated cooling system control
- Simple IoT workflow implementation

## Technologies Used
- Python
- Random Module
- Time Module

## Project Structure

DecodeLabs_Task4/

## How It Works
1. The program generates random temperature readings.
2. The temperature value is compared with the defined limit.
3. If the temperature is high, an alert is generated.
4. The cooling system is automatically switched ON.
5. If the temperature is normal, the system remains OFF.

## How to Run

1. Clone the repository
git clone 

2. Open the project folder
cd DecodeLabs_Task4

3. Run the program
python automation.py
